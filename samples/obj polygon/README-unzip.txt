EMA format is like an Wavefront OBJ file. You need to unzip
crypto.ema file in data directory for this program to work.
See below. An OBJ file is used instead of an EMA file.

OBJ format is partially implemented Wavefront OBJ file,

You need to unzip 3d-printer-web-test.zip to *.obj. It is
a text file format, a Wavefront OBJ file. If you use
cube.obj, you might want to increase the zoom scale.
